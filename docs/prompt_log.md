# 프롬프트 로그 (prompt_log.md)

## 1. 프로젝트명
전환교육 공백 분석 및 맞춤 경로 추천 시스템 (작동형 웹 프로토타입)

## 2. 작성 목적
본 문서는 **교육공공데이터 공모전 제출 시 생성형 AI 활용의 투명성을 보고하기 위한 보조자료**입니다. 본 프로토타입의 어떤 부분이 AI를 통해 만들어졌고, 어떤 검토를 거쳐 반영되었는지를 외부 검증 가능한 형태로 기록합니다. 본 문서는 발표자료 14쪽(윤리·검증 절차)과 직접 연결됩니다.

## 3. PPT와 웹 프로토타입의 역할 구분

> **본 웹 프로토타입은 15쪽 발표자료 전체를 웹사이트로 옮긴 것이 아닙니다.**
>
> 발표자료(15쪽 PPT)는 문제 정의, 데이터 설명, 차별성, 시스템 구조, 현장 적용, 윤리 검증, 기대효과를 설명하는 자료로서 그대로 유지됩니다. 본 웹 프로토타입은 그중 **실제 서비스 제공 화면만** 작동형으로 구현해, PPT에서 설명한 서비스가 실제로 동작하는 모습을 시연합니다.

| 현장 적용 시나리오·윤리 검증 절차·기대효과 설명 | 발표자료 (PPT) |

## 6. 초기 생성 프롬프트 요약
- **입력**: 발표자료 7·9·10·11·12쪽 화면 명세, 데이터 스키마 초안, 디자인 가이드(공공기관 톤·남색 포인트·카드형 레이아웃)
- **요구사항**: PPT 전체를 대체하지 않고 "서비스가 실제 작동하는 것처럼 보이는 핵심 화면"만 구현
- **AI 출력**: 5개 메뉴 작동형 SPA 풀 코드, JSON 스키마 4개 권역 더미 데이터, 추천 풀 6건
- **팀 검토 사항**: 컴포넌트 분리, 시각 위계, 배지 노출 위치, 추천 로직 단순화

## 7. 7쪽 보강 프롬프트 요약
- **핵심 강조점**: "현재 공백지수는 주지표, 추세위험도는 보조지표"라는 시각적 위계
- **AI 출력**: 공백지수 카드를 navy 배경 큰 카드로 분리, 추세위험도는 일반 흰색 카드로 축소
- **차트 요건**: Recharts LineChart, X·Y축·Tooltip·Legend 포함, "정책 판단용 추세 데이터" 라벨

## 8. 데이터 전처리 프롬프트 요약
- **입력**: 가상 CSV(transition_support.csv), 6개 수치 컬럼
- **AI 출력**: 가중합 기반 공백지수 산식, 5년 자원량 변화율 기반 추세위험도, 가장 부족한 항목 기반 공백 유형 분류
- **안전장치**: "프로토타입용 예시 산식" 주석, 실제 적용 시 교육청·특수교육 전문가·데이터 전문가 검토 필요 주석

## 9. 생성형 AI 활용 시 유의사항

### 9.1. 시연용 더미 데이터와 실제 공공데이터 구분
- 본 프로토타입의 모든 수치(공백지수·추세위험도·yearlySupport)는 **시연용 더미 데이터**입니다.
- 모든 화면 우측 상단에 "**시연용 더미 데이터**" 배지가 노출되도록 강제했습니다.
- 실제 공모전 발표나 후속 시범 적용 시 공공데이터(교육부·시도교육청·교육공공데이터포털)로 교체해야 하며, 교체 후에는 산식·임계값·가중치를 다시 검증해야 합니다.
- 더미 데이터로 도출된 결론(예: "서울 A권역은 프로그램 부족형")을 실제 정책·상담 판단에 그대로 사용하지 않습니다.

### 9.2. AI 산출물은 초안이며, 교사·담당자 검토 후 활용
- 모든 AI 생성 텍스트(교사용 상담 요약문, 학생·학부모 안내문, 교육청 정책 참고문)는 **초안**으로만 사용합니다.
- 모든 산출물 카드 하단에 "**AI가 생성한 초안이며, 교사 및 담당자의 검토 후 활용해야 합니다**" 문구가 항상 노출되도록 구현했습니다.
- 추천 카드 하단에도 "**교사 검토 후 학생·보호자 상담에 활용**" 안내가 노출됩니다.
- 5단계 검증 절차(데이터 기준연도 확인 → 추천 근거 확인 → 교사 검토 → 학생·보호자 상담 → 현장 적용 결과 피드백)를 거친 뒤에야 실제 활용 가능합니다.

### 9.3. 개인정보 보호 원칙
- 본 프로토타입은 백엔드가 없으며, 학생 프로필 입력값을 **저장하지 않습니다**. 새로고침 시 초기화됩니다.
- AI 도구(Claude 등)에 학생 개인 식별이 가능한 입력값을 직접 전달하지 않습니다.
- 화면에 표시되는 모든 학생 프로필 예시는 가공된 시연용 값입니다.

### 9.4. 자료 추가 시 기록 의무
- AI 도구를 사용하여 추가 산출물을 생성할 경우, 본 표(섹션 5)에 일자·도구·모델·목적·반영 여부를 즉시 추가합니다.
- 산식·가중치를 변경한 경우 [data_preprocessing_log.md](data_preprocessing_log.md)에도 함께 기록합니다.
